# 🤖 Grupo 6 Contra 1 - Robô Manivela

## 📌 Descrição do Projeto
Este projeto consiste no desenvolvimento de um protótipo utilizando o kit **LEGO Mindstorms EV3**, programado em **Python**, aplicando os conhecimentos adquiridos na disciplina de robótica.

O **Robô Manivela** foi concebido com o objetivo de proporcionar entretenimento e diversão. Seu nome é inspirado no personagem icônico e carismático do filme *Robôs* (2005).  

Diferente de sua referência, o aspecto cômico do robô não está em expressões ou comportamentos aleatórios, mas sim em sua **dança divertida e bastante engraçada**.
